The ActiveCGM graphics browser from Corel/Micrografx is required to view the graphics within ISIS

  • If you get an 'X' in the upper left corner of the box, or have never been in this part of ISIS before, click below to download the browser.

If ActiveCGM has previously been loaded on this PC, you may need to uninstall it before doing the reload.
Go to a Command (DOS) prompt.
Go to the main windows drive. Type "c:" if you are not already at the "c" prompt.
Change directories to the Windows directory. It probably is "cd winnt". For Windows 95/98 it may be "cd windows/system"
Type "regsvr32 /u acgm.dll"
Wait a couple of seconds and you should get a message that the control was unregistered.

Then...

Download this control (version 7.1 SP02) for all Windows operating systems now.

You may get the following message(s) when loading the control.


Click Yes to load the control.


Click Yes if you would like to be able to view TIF files by double-clicking on them in a directory list. ISIS doesn't depend on this choice.
