iService DVD July, 2008 Update

The July, 2008 update to the iService DVD continues the enhancements to the completely new viewing system which was introduced on the January, 2001 CD. The iService DVD is Internet Explorer 7 compatible. This will make it easy to link to different types of information. Much of the content has been cleaned up.

Also, there is an update to the ActiveCGM control that is used to display the graphics.
You only need the update if you are running on Windows XP.
The update is at the bottom of this page.
